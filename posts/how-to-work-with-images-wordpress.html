How to Work with Images in WordPress


One of the most common problems you face working in WordPress is what to do about images. In fact, the WordPress API authors haven't done a good job exposing the right functions to use in this situation. In this post, I will show you how to display the image you want, 100% of the time!

The first thing you need to remember is that you can use the_content(), which will display HTML, including <img /> tags. It will display images in the order than the post author inputted them, so theme designers often use it in the single page. When we use the_content(), we're handing control of the page flow to the theme author.

But that's usually not enough control. Let's look at two more techniques. The first one is called the Feature Image technique, and it lets post authors directly assosiate an image with a post. Then, you can use a template tag to display that image.
